    Do Now:

    We will try out a new way to review what we have learned using GimKit, when finished

    Fill out the form for any CSS code that you want me to review or cover as a class on block day. Form closes on Tuesday at end of day.


    LESSON/ACTIVITY:
    Continue working on the challenges for CSS in FreeCodeCamp, if you are finished either start working on your website for CSS, play CSS Grid Garden, or Flexbox Froggy. If you have finished that then start working on the CSS for your pages.

    Quiz:
    WHAT: HTML code, semantic elements, Using external CSS for fonts, sizing, images, boxes, borders, and shading
    WHEN: 11-15NOV2019

    Quiz Retakes:
    Quiz retakes can be found in the Quiz/Test area. If offered, they can be retaken once and will close at the end of the quarter. Be careful as not being prepared and taking the quiz again can hurt your quiz average worse than not retaking at all.


    PROJECTS:

    1st Quarter:
    PAST LATE ACCEPTANCE - DUE 23AUG2019 - Hello World Project, Syllabus Walkthrough
    PAST LATE ACCEPTANCE - DUE 30AUG2019 - BrainStorm Session 1
    PAST LATE ACCEPTANCE - DUE 06SEP2019 - FreeCodeCamp - Basic HTML and HTML5
    PAST LATE ACCEPTANCE - 7th Period only DUE 17SEP2019 - My Website Progress Check 1 - HTML (2 Pages)
    PAST LATE ACCEPTANCE - DUE 22SEP2019 - My Website Progress Check 2 - HTML (4 Pages)
    DUE 13OCT2019 - FreeCodeCamp - Introduction to Basic CSS


    2nd Quarter:
    DUE 27OCT2019 - My Website Progress Check 3 - CSS (External CSS for fonts, NOT positioning)
    DUE 10NOV2019 - Review of HTML and CSS for quiz
    DUE 11-15NOV2019 - Quiz covering HTML code, semantic elements, Using external CSS for fonts, sizing, images, boxes, borders, and shading
    DUE 15NOV2019 - My Website Progress Check 4 - HTML and CSS for Accessibility
    DUE 05DEC2019 - My Website Progress Check 5- Aesthetically pleasing website using CRAP acronym
    DUE 14DEC2019 - Final Website Progress Check - Responsive for all screen sizes


    **Remember to research, ask a neighbor, then put your flag up if you need help.

    **LEGEND:

    DUE - The assignment is due at 11:59 pm of the due date of the assignment.

    LATE - 20 % markdown on assignment. Only accepted 2 days after original assignment due date unless I already talked to you about it.

    PAST LATE ACCEPTANCE - unless I have already talked to you, you will receive a max credit for the assignment of 50% if you have legitimately met the requirements otherwise it is a 0% as you are past the late turnin date.

    NO LONGER ACCEPTED - Not accepted for any credit.

    **Projects will only keep coming, if you are running behind then you should probably schedule a time with me on Wednesday morning before class (if I am available) to get caught up. Use the form in the administrative section. Alternatively, you are able to work at home using source control and Github. The administrative section has video instructions for downloading the required software.
